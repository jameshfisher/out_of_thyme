# Out of Thyme

```
GET  /
GET  /rack
GET  /rack/:herb
POST /rack/:herb with body "empty" or "full"
```

We support the following herbs:

```
satureja - thyme
origanum - origano
salvia - sage
mentha - mint
petroselinum - parsley
cichorium - chicory
carum - caraway
melissa - (lemon) balm
lavendula - lavender
levisticum - lovage
calendula - marigold
hypericum - st. john's wort
tanacetum - tansy
linium - flax
```
